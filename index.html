실습 중심

읽기만 하는 학습보다 직접 코드를 작성하고 실행 결과를 확인하는 흐름에 초점을 맞췄습니다.

초보자 친화

처음 시작하는 사람도 겁먹지 않도록 쉬운 언어와 명확한 구성으로 안내합니다.

무엇을 배우게 되나요?

문법 설명에 그치지 않고, 사고 방식과 구현 경험이 함께 쌓이도록 학습 흐름을 설계했습니다.

기초 문법

변수, 자료형, 조건문, 반복문, 함수처럼 모든 언어에서 반복해서 만나는 핵심 개념을 익힙니다.

문제 해결

문제를 작은 단계로 나누고, 입력과 출력의 규칙을 정리하며 해결 절차를 코드로 표현합니다.

프로그램 설계

자료구조와 흐름 제어를 활용해 계산기, 텍스트 처리기, 간단한 게임처럼 동작하는 프로그램을 만듭니다.

이런 프로젝트를 만들어요

작게 시작해도 프로그래밍의 재미를 느낄 수 있도록 단계별 프로젝트를 준비했습니다.

Starter 숫자 맞히기 게임

입력값을 비교하고 조건에 따라 다른 결과를 출력하며 프로그램의 기본 흐름을 익힙니다.

Practice 성적 계산기

반복문과 함수를 사용해 여러 데이터를 계산하고 정리하는 과정을 연습합니다.

Portfolio 미니 도구 모음

텍스트 변환기, 단위 계산기, 간단한 퍼즐처럼 직접 만든 프로그램을 모아 나만의 결과물로 정리합니다.
